## GitHub Assignment
- I learned the steps of using git to push and pull code on my local and remote repo.
Here is a simple list of the steps of using Git.
1. git add [filename]
2. git commit -m "what you did"
3. git push
- For getting changes from your remote repo:
1. git fetch
2. git status
3. git pull
